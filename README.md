## 🖤 The Descent Begins
**Monstropoly** is a real-time multiplayer board game where survival is the only currency that matters. Ritualists must navigate a 40-space descent into the void, making blood pacts, manifesting gravestones, and desecrating lands to bankrupt their rivals' souls.

Built with a focus on visceral atmospheric immersion and sub-second state synchronization, Monstropoly transforms the familiar board game formula into a haunting, high-stakes ritual.

## 🛠️ The Architect's Tools
*   **Frontend**: React & TypeScript for a robust, type-safe ritual engine.
*   **Styling**: Vanilla CSS & Tailwind for a premium gothic aesthetic (Blood, Smoke, Parchment).
*   **State Sync**: Firebase Realtime Database handles sub-second mirroring of the ritual state across all participants.
*   **Chronicle Intelligence**: Google Gemini Pro generates procedural narrative logs, translating game events into atmospheric ritual chronicles.
*   **Authentication**: Firebase Auth ensures only sanctioned souls can enter the void.

## 💀 Ritual Features
*   **The AnteChamber**: A real-time lobby system with host controls, player presence tracking, and automated ethereal cleanup.
*   **Crimson Ritual Visuals**: Layered, animated blood drips and drifting spectral fog that intensify as the game progresses.
*   **Strategic Manifestation**: A refined property system where building (Gravestones/Mausoleums) is turn-based and restricted by the purity of the color group.
*   **Diplomatic Blood Pacts**: A complex real-time trading system for exchanging Souls, Properties, and rare Artifacts.
*   **Ancestral Artifacts**: Unique items like the **Skeleton Key** that provide deterministic escapes from the **Pit of Despair**.

## 📖 How to manifest (Play)
1.  **Enter the Gateway**: Sign up or log in at the Manifestation Gateway.
2.  **Summon a Pact**: Create a lobby or join one using a unique Pact Code.
3.  **The Casting**: Roll the bone dice to move through the void.
4.  **The Resolution**: Buy unclaimed haunts, pay "Soul Tolls" to rivals, or draw from the decks of **Torment** and **Omen**.
5.  **Dominance**: The last ritualist with souls remaining becomes the master of the void.

## 🚀 Local Manifestation
**Prerequisites**: Node.js & a burning desire for survival.

1.  **Clone the Ritual**:
    ```bash
    git clone https://github.com/deveraxtianrey-cyber/MONOPOLY.git
    cd MONOPOLY
    ```
2.  **Install Offerings**:
    ```bash
    npm install
    ```
3.  **Divine Configuration**:
    Create a `.env.local` or `.env` file with your **Firebase** and **Gemini** credentials.
4.  **Ignite the Engine**:
    ```bash
    npm run dev
    ```
